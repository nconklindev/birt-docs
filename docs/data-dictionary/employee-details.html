Employee Details

The Employee Details entity provides data columns that are attributes or descriptive information that are associated with an employee.

Supported Types​

EmployeeBusiness StructureEmployee Time-SeriesBusiness Structure Time-SeriesWork Unit

Data Granularity​

Provides one row per employee.
